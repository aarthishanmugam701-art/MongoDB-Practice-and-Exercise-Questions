# MongoDB-Practice-and-Exercise-Questions

# MongoDB CRUD Operations Project

## Objective
To understand MongoDB basics and perform CRUD operations using Node.js, Express, and Mongoose.

---

## 1. Database & Collection Setup
- Database created: studentDB
- Collection created: students

---

## 2. Insert Operations
- Insert one document into students collection
- Insert multiple documents with fields:
  - name
  - age
  - course
  - status

---

## 3. Read Operations
- Fetch all student documents
- Filter students based on course (e.g., MERN Stack)

---

## 4. Update Operations
- Update single student (change status to completed)
- Update multiple students at once

---

## 5. Delete Operations
- Delete one document based on condition
- Delete all documents for practice

---

## 6. Query Operators Used
- $gt → greater than
- $lt → less than
- $in → match multiple values
- $and → multiple conditions
- $or → either condition
- $exists → check field exists

---

## 7. Use Case
Library Management System / E-Commerce Product System:
- Insert books/products
- Update stock/price
- Search using filters

---

## Outcome
By completing this task, I learned:
- MongoDB CRUD operations
- Query operators
- Database and collection structure
- Real-world data handling
